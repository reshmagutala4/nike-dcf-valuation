# nike-dcf-valuation
## Overview
Built a 5-year Discounted Cash Flow (DCF) model in Excel to estimate the intrinsic value of Nike.
## Skills Demonstrated
- Financial Modeling
- Corporate Finance
- Forecasting
- Valuation Analysis
## Model Components
- Revenue forcasting
- Operating margin estimation
- Free Cash Flow calculation
- Terminal value estimation
- Enterprise value and equity value computation
## Key Assumptions
- Weighted Average Cost of Capital (WACC)
- Revenue growth rate
- Terminal growth rate

- nike-dcf-model.xlsx
